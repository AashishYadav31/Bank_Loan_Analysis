📊 Bank Loan Analysis & Dashboard Project

This project focuses on analyzing bank loan data to generate actionable insights using Python, SQL, and Power BI. It aims to monitor lending performance, evaluate loan quality, and support data-driven decision-making in financial institutions.

🚀 Project Overview

The objective of this project is to build a comprehensive Bank Loan Report System that tracks key metrics such as loan applications, funded amounts, repayments, interest rates, and borrower risk profiles. The project integrates data processing, querying, and visualization to provide a complete analytical solution.

The analysis helps in:

Assessing loan portfolio health
Identifying good vs bad loans
Tracking monthly trends and KPIs
Understanding borrower behavior and risk

Based on the problem statement, the system includes multiple KPIs like Total Loan Applications, Funded Amount, Amount Received, Average Interest Rate, and Debt-to-Income Ratio (DTI) .

🛠️ Tech Stack

Python 🐍

Used for data cleaning, preprocessing, and exploratory data analysis (EDA).

SQL 🗄️

Used for data extraction, transformation, KPI calculations, and querying large datasets efficiently.

Power BI 📈

Used to create interactive dashboards and visualizations for business insights.
